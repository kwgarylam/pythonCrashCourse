### Data type and variable
|Data type|variable|
|---|----|
|Number|int, float, bool|
|String|str, chr|
|Container|list, dict, , set, tuple|

#### Examples

#### Interger
  ```python
  x = 1
  print(x)
  print(type(x))
  ```

#### Float
  ```python
  x = 1.5
  print(x)
  print(type(x))
  ```
